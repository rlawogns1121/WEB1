CSS란 무엇인가?

Cascading Style Sheets is a styhle sheet language used for described the presentation of a document written in a markup language like HTML [1] CSS is a cornerstone technology of a document written in a markup languge, HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page. It provides a means to create structed documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes and other items. HTML elements are delineated by tags, written using angle brackets.
